Example static file

You can add files to the content directory and have them passed through unchanged to the browser.

By default the following extensions are passed through:

  • htm
  • html
  • css
  • js
  • gif
  • jpg
  • jpeg
  • png
  • svg
  • bmp
  • webp
  • pdf

If you would like to add more, add them to _glue-site/config/
